# 📚 Projeto Biblioteca

Este é um projeto desenvolvido para a faculdade com o objetivo de simular o funcionamento de uma biblioteca digital. O sistema permite listar livros, fazer buscas e exibir informações utilizando tecnologias modernas do front-end.

## 🔧 Tecnologias Utilizadas

- [Vue.js](https://vuejs.org/) – para construção de componentes reativos
- [Axios](https://axios-http.com/) – para simulação de requisições HTTP
- [Bootstrap 5](https://getbootstrap.com/) – para estilização e responsividade
- HTML5 e CSS3 puros
  
## ✨ Funcionalidades

- ✅ Listagem de livros
- ✅ Busca de livros
- ✅ Layout responsivo com Bootstrap
- ✅ Integração com Vue.js para comportamento dinâmico
